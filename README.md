# 水质监测模型对比研究 Water-Quality-Prediction
## 一、项目背景 本项目面向**水利物联网水质智能监测**场景，针对水体安全监测需求，基于公开水质数据集构建机器学习预测模型。通过对比7种经典算法的性能表现，筛选最优水质分类模型，实现水质安全状态自动判定，为水利环境监测提供智能化算法支撑。 
## 二、数据集 - **来源**：Kaggle 公开水质监测数据集 - **用途**：水质安全二分类预测（安全/污染） - **特征**：包含pH值、溶解氧、浊度等多项水质关键理化指标 - **规模**：标准化水质监测样本，适用于机器学习模型训练与验证 
## 三、技术栈 - 编程语言：Python - 数据处理：Pandas、NumPy - 可视化分析：Matplotlib、Seaborn - 机器学习框架：Scikit-learn - 核心算法：7种经典分类模型 
## 四、实现流程 1. 数据加载与探索性数据分析（EDA） 2. 数据预处理：缺失值填充、异常值处理、数据标准化 3. 数据集划分（训练集/测试集） 4. 7种机器学习模型搭建、训练与调优 5. 模型性能评估与对比分析 6. 结果可视化与最优模型筛选 
## 五、核心模型（7种对比） 1. Logistic Regression（逻辑回归） 2. Decision Tree（决策树） 3. Random Forest（随机森林） 4. XGBoost 5. KNeighbors（K近邻）、 6. SVM（支持向量机） 7. AdaBoost
## 六、实验结果 - 完成7种模型的准确率、精确率、召回率等核心指标对比 - **最优模型**：SVM（支持向量机）
